# SliceScout

SliceScout is a pizza-themed local ordering app that helps you find the best
closest pizza place and jump to an order page.

## Features

- Browser geolocation for "near me" pizza discovery
- OpenStreetMap-powered pizza place lookup when location access is available
- Distance and order-availability scoring for a highlighted best match
- Pizza-themed marketplace UI with ranked alternatives
- Demo cities and fallback pizzerias for quick testing

## Run locally

```bash
npm start
```

Then open <http://localhost:4173>.

## Verify

```bash
npm run check
```
