# SliceApp

Pizza-focused food delivery web app inspired by DoorDash.

## Run locally

```bash
cd pizzadash
npm install
npm run dev
```

## What it does

- Uses your browser location (or demo fallback) to find nearby pizza places
- Scores places based on closeness and pizza-ordering signals
- Highlights the best closest option first
- Links each result directly to an order page (restaurant site or DoorDash search)
