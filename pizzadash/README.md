# SliceSprint (PizzaDash)

Pizza-themed, DoorDash-style app that finds nearby pizza places and links you to order pages.

## Features

- **Near-me search** using browser geolocation
- **Live pizza data** fetched from OpenStreetMap Overpass API
- **Best closest recommendation** with a pizza score + delivery ETA estimate
- **Order links** that prefer a direct restaurant website and fall back to DoorDash search
- **Radius controls** to tune search range

## Tech stack

- React 19
- TypeScript
- Vite

## Quick start

```bash
npm install
npm run dev
```

Open the app URL from Vite (typically `http://localhost:5173`) and click **Use my location**.

## Build

```bash
npm run build
```
